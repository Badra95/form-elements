Input Type radio

L'élément HTML <input> est utilisé pour créer un contrôle interactif dans un formulaire web qui permet à l'utilisateur de saisir des données. L'élément HTML <input type="radio"> un bouton radio. L'attribut value doit alors être utilisé pour définir la valeur associé à cet élément.

Exemple

j'accepte les conditions
je refuse les conditions

Code derrière tout ça :

<input type="radio">
