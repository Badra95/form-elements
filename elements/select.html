Select

L'élément HTML <select> est utilisé pour créer un menu.

Exemple

Code derrière tout ça :

<select>
<option> .... <option>
<option> php <option>
<option> j.s <option>
<option> jquery <option>
<option> ruby <option>
<select>

Liste des éléments :

  • input type checkbox
  • input type email
  • input type file
  • input type number
  • input type radio
  • input type range
  • input type reset
  • input type submit
  • input type text
  • select
