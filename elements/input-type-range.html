Input Type range

L'élément HTML <input> est utilisé pour créer un contrôle interactif dans un formulaire web qui permet à l'utilisateur de saisir des données. L'élément HTML <input type="range"> permet de saisir un nombre dont la valeur exacte n'est pas importante.

Exemple


Code derrière tout ça :

<input type="range" >
