Input Type Email

L'élément HTML <input type="email"> sont utilisés pour permettre à l'utilisateur d'entrer ou de modifier une adresse e-mail, et même d'y intégrer une liste d'adresses e-mail. <input type="text"> permet d'ajouter du texte sur une seule ligne.

Exemple




Code derrière tout ça :

<input id="emailAddress" type="email" placeholder="Un Email">
<input id="emailAddress" type="email" placeholder="Plusieurs Emails" multiple>
<input type="email" size="10" placeholder="10 caractères">

Quelques attributs :

  • placeholder : Une indication, destinée à l'utilisateur, pour ce qui peut être saisi dans le champ.
  • multiple : Si l'attribut multiple est spécifié, la valeur peut être une liste d'adresses électroniques correctement formées séparées par des virgules.
  • size : Modifier la taille de l'élement en spécifiant le nombre de caractères que la boîte de saisie peut afficher
