Input Type file

L'élément HTML <file> permettent à l'utilisateur de modifier les champs files contenant la liste des fichiers sélectionnés. Par défaut un seul fichier est accepté à moins que l'élément contienne l'attribut multiple. Une fois l'élément cliqué, une fenêtre système affiche un menu de sélection permettant l'ajout de fichiers depuis le système ou la création d'un nouveau fichier à la volée, en prennant par exemple une photo avec la caméra de l'ordinateur.

Exemple



Code derrière tout ça :

<input type="file">
<input type="file" multiple>

Quelques attributs :

  • multiple : Si l'attribut multiple est spécifié, la valeur peut être une liste d'adresses électroniques correctement formées séparées par des virgules.
